display "append" behavior
Append() is very similar to after() with one major difference. The difference is that after() inserts code after the end of the specified object(s) (after the closing tags). Append() inserts text BEFORE the end of the specified object. To insert after the start of the object, use prepend();
display "html" behavior
The HTML method can either get html code from an object (if no parameter is given) or sets the HTML of ALL matching objects (the replacement HTML is the parameter for html()).
display "attr" behavior
attr() is used to either get or set an attribute for a specified object. Keep in mind, this needs to go through the DOM object root for whatever you are modifying. For example, ("style", "background-color: green").
display "val" behavior
Val() is the same kind of method as attr() and HTML(), instead focussing on the "value" of something else. (Confirmed to work for entries in forms/inputs.)
